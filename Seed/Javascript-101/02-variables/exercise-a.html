Exercise 2a: Create variables for personal information

In this exercise, you'll practice creating and using variables to store personal information. Create variables for your name, age, and favorite color, then display them using console.log().

Use appropriate variable names and data types for each piece of information.

// Create your variables here // Display the information using console.log() console.log("My name is:"); console.log("I am years old"); console.log("My favorite color is:");

Remember to use let or const to declare your variables.

Use string data type for name and favorite color, and number for age.

You can use the + operator to combine strings and variables in console.log().
