2. Variables and Data Types

Theory

Variables are used to store data that can be used and manipulated throughout your program. Think of variables as containers that hold information. In JavaScript, you can create a variable using the var, let, or const keywords.

Data types specify the kind of data that can be stored in a variable. JavaScript has several data types, including:

  • String: A sequence of characters, e.g., "Hello, World!"
  • Number: Numeric values, e.g., 42 or 3.14
  • Boolean: Represents true or false
  • Object: A collection of key-value pairs
  • Array: An ordered list of values
  • Null: Represents the intentional absence of any object value
  • Undefined: A variable that has been declared but not assigned a value

Example: Declaring Variables

Let's look at an example of how to declare and use variables in JavaScript:


let name = "Alice";
let age = 25;
let isStudent = true;

console.log(name); // Outputs: Alice
console.log(age); // Outputs: 25
console.log(isStudent); // Outputs: true

In this example:

  • let name = "Alice"; declares a variable named name and assigns it the string value "Alice".
  • let age = 25; declares a variable named age and assigns it the number value 25.
  • let isStudent = true; declares a variable named isStudent and assigns it the boolean value true.
  • console.log() is used to display the values of the variables in the console.

When you run this code, you'll see the values of the variables displayed in the console.
