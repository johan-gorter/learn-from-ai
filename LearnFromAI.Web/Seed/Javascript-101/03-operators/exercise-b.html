Exercise 3b: Determine if a number is even or odd

In this exercise, you'll use the modulus operator (%) to determine if a number is even or odd. The modulus operator returns the remainder of a division operation.

Your task is to write a JavaScript program that takes a number, uses the modulus operator to check if it's even or odd, and then uses console.log() to display the result of the modulus operation.

Hint: When a number is divided by 2, an even number will have a remainder of 0, while an odd number will have a remainder of 1.

Use the modulus operator (%) to find the remainder when dividing by 2.

The expression will be: number % 2

Use console.log() to display this result.
