Exercise 4b: Determine a student's grade

In this exercise, you'll use conditional statements to determine a student's grade based on their score.

Your task is to write a JavaScript program that takes a student's score (0-100) and uses if-else statements to determine the corresponding letter grade. Then, use console.log() to display the grade.

Use the following grading scale:

  • A: 90-100
  • B: 80-89
  • C: 70-79
  • D: 60-69
  • F: 0-59

Hint: You'll need to use multiple if-else statements to cover all grade ranges.

Use if-else statements to check the following conditions:

  • If the score is 90 or above, it's an A
  • If the score is 80 or above (but less than 90), it's a B
  • If the score is 70 or above (but less than 80), it's a C
  • If the score is 60 or above (but less than 70), it's a D
  • If the score is below 60, it's an F

Use console.log() to display the appropriate grade for each case.
