Exercise 4a: Check if a number is positive, negative, or zero

In this exercise, you'll use conditional statements to determine if a number is positive, negative, or zero.

Your task is to write a JavaScript program that takes a number and uses if-else statements to check its sign. Then, use console.log() to display whether the number is positive, negative, or zero.

Hint: You'll need to use multiple if-else statements to cover all cases.

Use if-else statements to check the following conditions:

  • If the number is greater than 0, it's positive
  • If the number is less than 0, it's negative
  • If the number is equal to 0, it's zero

Use console.log() to display the appropriate message for each case.
