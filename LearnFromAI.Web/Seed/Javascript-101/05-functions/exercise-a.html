Exercise 5a: Write a function that converts Celsius to Fahrenheit

Your task is to create a function named celsiusToFahrenheit that takes a temperature in Celsius as input and returns the equivalent temperature in Fahrenheit.

The formula for conversion is: F = (C * 9/5) + 32

After creating the function, use it to convert 25°C to Fahrenheit and log the result.

Remember the formula: F = (C * 9/5) + 32

Use the return keyword to send back the calculated Fahrenheit temperature.

Don't forget to use parentheses to ensure the correct order of operations!
